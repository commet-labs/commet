## Initial Setup

```bash
pnpm add -g commet
commet login
commet link
commet pull
```

## Daily Development

After adding types in the dashboard:

```bash
commet pull
```

## Switch Organizations

```bash
commet switch
```

## Switch Environments

```bash
commet logout
commet login  # Select different environment
commet link
```
