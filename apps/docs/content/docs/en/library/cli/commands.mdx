# CLI Commands

## Authentication

### `commet login`

Authenticate with Commet using OAuth device flow.

```bash
commet login
```

Prompts for environment (Sandbox/Production), opens browser, stores credentials in `~/.commet/auth.json`.

---

### `commet logout`

```bash
commet logout
```

Removes `~/.commet/auth.json`.

---

### `commet whoami`

```bash
commet whoami
```

Shows authentication status and linked organization.

---

## Project Management

### `commet link`

Link project to a Commet organization.

```bash
commet link
```

Creates `.commet` file with organization config.

---

### `commet unlink`

```bash
commet unlink
```

Removes `.commet` file.

---

### `commet switch`

Switch to a different organization.

```bash
commet switch
```

Updates `.commet` with new organization (same environment).

---

### `commet info`

```bash
commet info
```

Shows current authentication and project status.

## Type Generation

### `commet pull`

Generate TypeScript types in `.commet/types.d.ts`.

```bash
commet pull
```

Creates:
- `.commet/` directory
- `.commet/types.d.ts` with event types, seat types, and products
- Updates `tsconfig.json` to include the types
- Updates `.gitignore` to ignore `.commet/`

**Generated file:**
```typescript
// .commet/types.d.ts
/**
 * Event types available in your organization:
 * - "api_call": API Calls - Track API usage
 * - "email_sent": Emails Sent - Email delivery events
 * 
 * Seat types available in your organization:
 * - "admin": Administrator - Full access (Free)
 * - "user": Standard User - Regular access
 * 
 * Products available in your organization:
 * - "prd_c2JV8SjRxSD2y6cRSMzYJPJ2": API Platform - Usage-based billing
 * - "prd_fnvJZwETRrYTmt9y42tLQEMH": Support Plan - Fixed monthly fee
 */
declare module '@commet/node' {
  interface CommetGeneratedTypes {
    eventType: "api_call" | "email_sent";
    seatType: "admin" | "user";
    productId: "prd_c2JV8SjRxSD2y6cRSMzYJPJ2" | "prd_fnvJZwETRrYTmt9y42tLQEMH";
  }
}
```

**Usage:**
```typescript
import { Commet } from '@commet/node';

const commet = new Commet({ apiKey: '...' });

// Autocomplete for event types
await commet.usage.create({
  customerId: 'cus_123',
  eventType: 'api_call', // âœ“ autocomplete
});

// Autocomplete for seat types
await commet.seats.add({
  customerId: 'cus_123',
  seatType: 'admin', // âœ“ autocomplete
  count: 5
});

// Autocomplete for products
await commet.subscriptions.create({
  customerId: 'cus_123',
  productId: 'prd_c2JV8SjRxSD2y6cRSMzYJPJ2', // âœ“ autocomplete
});
```
