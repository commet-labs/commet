# Installing Commet CLI

Requires Node.js 18+.

## Install

<Tabs items={['pnpm', 'npm', 'yarn']}>
<Tab value="pnpm">
```bash
pnpm add -g commet
```
</Tab>
<Tab value="npm">
```bash
npm install -g commet
```
</Tab>
<Tab value="yarn">
```bash
yarn global add commet
```
</Tab>
</Tabs>

Verify:

```bash
commet --version
```

## Setup

<Steps>
<Step>

### Authenticate

```bash
commet login
```

Select environment (Sandbox/Production) and authenticate in browser.

</Step>

<Step>

### Link Project

```bash
cd your-project
commet link
```

Select your organization.

</Step>

<Step>

### Generate Types

```bash
commet pull
```

Creates `.commet/types.d.ts` with your event and seat types.

</Step>
</Steps>
