<Callout type="info">
**Prerequisites:** Customer must exist and have seat types configured.
</Callout>

## Add Seats

```typescript
const result = await commet.seats.add({
  customerId: 'cus_xxx',
  seatType: 'editor',
  count: 5
})
```

<Callout type="info">
**Customer Identification**: Use `customerId` or `externalId` to identify customers.
</Callout>

## Remove Seats

```typescript
const result = await commet.seats.remove({
  customerId: 'cus_xxx',
  seatType: 'editor',
  count: 2
})
```

## Set Exact Count

```typescript
const result = await commet.seats.set({
  customerId: 'cus_xxx',
  seatType: 'editor',
  count: 10
})
```

## Using External IDs

```typescript
const result = await commet.seats.add({
  externalId: 'stripe_cus_abc123',
  seatType: 'editor',
  count: 5
})
```

## Bulk Update

Update multiple seat types at once:

```typescript
const result = await commet.seats.bulkUpdate({
  customerId: 'cus_xxx',
  seats: {
    admin: 3,
    editor: 50,
    viewer: 100
  }
})
```

## Get Balance

```typescript
const balance = await commet.seats.getBalance({
  customerId: 'cus_xxx',
  seatType: 'editor'
})

console.log(`Current seats: ${balance.data.current}`)
```

## Get All Balances

```typescript
const allBalances = await commet.seats.getAllBalances({
  customerId: 'cus_xxx'
})

console.log(allBalances.data)
// { editor: { current: 50, asOf: '...' }, viewer: { current: 100, asOf: '...' } }
```

## List Events

```typescript
const events = await commet.seats.listEvents({
  customerId: 'cus_xxx',
  seatType: 'editor',
  limit: 50
})

// Pagination
if (events.hasMore) {
  const nextPage = await commet.seats.listEvents({
    customerId: 'cus_xxx',
    cursor: events.nextCursor
  })
}
```
