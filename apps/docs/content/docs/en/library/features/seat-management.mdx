---
title: Seats
description: Manage seat-based licenses
---

## Add Seats

```typescript
await commet.seats.add({
  externalId: 'user_123',
  seatType: 'editor',
  count: 5
})
```

## Remove Seats

```typescript
await commet.seats.remove({
  externalId: 'user_123',
  seatType: 'editor',
  count: 2
})
```

## Set Exact Count

```typescript
await commet.seats.set({
  externalId: 'user_123',
  seatType: 'editor',
  count: 10
})
```

## Set All Seat Types

Update multiple seat types at once:

```typescript
await commet.seats.setAll({
  externalId: 'user_123',
  seats: {
    admin: 3,
    editor: 50,
    viewer: 100
  }
})
```

## Get Balance

```typescript
const balance = await commet.seats.getBalance({
  externalId: 'user_123',
  seatType: 'editor'
})

console.log(`Current seats: ${balance.data.current}`)
```

## Get All Balances

```typescript
const balances = await commet.seats.getAllBalances({
  externalId: 'user_123'
})

console.log(balances.data)
// { editor: { current: 50, asOf: '...' }, viewer: { current: 100, asOf: '...' } }
```

## Customer Identification

Use `customerId` or `externalId`:

```typescript
// By Commet ID
await commet.seats.add({
  customerId: 'cus_xxx',
  seatType: 'editor',
  count: 5
})

// By your system's ID
await commet.seats.add({
  externalId: 'stripe_cus_abc123',
  seatType: 'editor',
  count: 5
})
```
