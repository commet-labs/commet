# Quick Start

Get your billing system running in 5 minutes.

<Callout type="info">
**Prerequisites:** Node.js 18+ and a [Commet account](https://sandbox.commet.co).
</Callout>

<Steps>
<Step>

## Install

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
<Tab value="npm">
```bash
npm install @commet/node
```
</Tab>
<Tab value="pnpm">
```bash
pnpm add @commet/node
```
</Tab>
<Tab value="yarn">
```bash
yarn add @commet/node
```
</Tab>
<Tab value="bun">
```bash
bun add @commet/node
```
</Tab>
</Tabs>

</Step>

<Step>

## Initialize

```typescript
import { Commet } from '@commet/node'

const commet = new Commet({
  apiKey: process.env.COMMET_API_KEY!,
  environment: 'sandbox', // or 'production'
})
```

</Step>

<Step>

## Track Usage

```typescript
await commet.usage.create({
  eventType: 'api_call',
  customerId: 'cus_xyz',
  idempotencyKey: `evt_${Date.now()}`,
  properties: [
    { property: 'endpoint', value: '/users' },
    { property: 'method', value: 'GET' }
  ]
})
```

</Step>

<Step>

## Manage Seats

```typescript
// Add seats
await commet.seats.add({
  customerId: 'cus_xyz',
  seatType: 'editor',
  count: 5
})

// Set exact count
await commet.seats.set({
  customerId: 'cus_xyz',
  seatType: 'editor',
  count: 10
})
```

</Step>

<Step>

## Create Subscription

```typescript
const subscription = await commet.subscriptions.create({
  customerId: 'cus_xyz',
  items: [
    { priceId: 'price_xxx', quantity: 1 }
  ]
})
```

</Step>
</Steps>
