# Quick Start

Get your billing system running in 5 minutes.

<Callout type="info">
**Prerequisites:** Node.js 18+ and a [Commet account](https://commet.co).
</Callout>

<Steps>
<Step>

## Install

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
<Tab value="npm">
```bash
npm install @commet/node
```
</Tab>
<Tab value="pnpm">
```bash
pnpm add @commet/node
```
</Tab>
<Tab value="yarn">
```bash
yarn add @commet/node
```
</Tab>
<Tab value="bun">
```bash
bun add @commet/node
```
</Tab>
</Tabs>

</Step>

<Step>

## Initialize

```typescript
import { Commet } from '@commet/node'

const commet = new Commet({
  apiKey: process.env.COMMET_API_KEY!,
  environment: 'sandbox', // or 'production'
})
```

</Step>

<Step>

## Subscribe to a Plan

Each customer can have one active subscription at a time.

```typescript
// Subscribe customer to a plan
const subscription = await commet.subscriptions.create({
  externalId: 'user_123',  // Your user ID
  planId: 'plan_pro',
  billingInterval: 'monthly',
  initialSeats: { editor: 5 }
})
```

</Step>

<Step>

## Track Usage

```typescript
await commet.usage.track({
  externalId: 'user_123',
  eventType: 'api_call',
  idempotencyKey: `evt_${Date.now()}`,
  properties: { endpoint: '/users', method: 'GET' }
})
```

</Step>

<Step>

## Manage Seats

```typescript
// Add seats
await commet.seats.add({
  externalId: 'user_123',
  seatType: 'editor',
  count: 5
})

// Set exact count
await commet.seats.set({
  externalId: 'user_123',
  seatType: 'editor',
  count: 10
})
```

</Step>
</Steps>
