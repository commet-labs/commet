## Crear

Idempotente con `externalId` - si el cliente existe, lo retorna en lugar de crear un duplicado.

```typescript
await commet.customers.create({
  email: 'billing@acme.com',
  externalId: 'user_123',
})
```

## Obtener

```typescript
const customer = await commet.customers.get('cus_xxx')
```

## Actualizar

```typescript
await commet.customers.update('cus_xxx', {
  email: 'new@acme.com',
})
```

## Listar

```typescript
const customers = await commet.customers.list()
```
