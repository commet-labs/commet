---
title: Eventos de Uso
description: Rastrea uso para facturación
---

El uso se agrega y factura al final del período.

## Track

```typescript
await commet.usage.track({
  externalId: 'user_123',
  eventType: 'api_call',
})
```

## Batch

Más eficiente para rastreo de alto volumen.

```typescript
await commet.usage.trackBatch({
  events: [
    { externalId: 'user_123', eventType: 'api_call' },
    { externalId: 'user_456', eventType: 'api_call' },
  ]
})
```
