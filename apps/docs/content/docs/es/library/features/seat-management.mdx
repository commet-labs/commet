title: Seats
description: Gestiona licencias por asiento
---

## Agregar Seats

```typescript
await commet.seats.add({
  externalId: 'user_123',
  seatType: 'editor',
  count: 5
})
```

## Remover Seats

```typescript
await commet.seats.remove({
  externalId: 'user_123',
  seatType: 'editor',
  count: 2
})
```

## Establecer Cantidad Exacta

```typescript
await commet.seats.set({
  externalId: 'user_123',
  seatType: 'editor',
  count: 10
})
```

## Establecer Todos los Tipos

Actualiza múltiples tipos de seat a la vez:

```typescript
await commet.seats.setAll({
  externalId: 'user_123',
  seats: {
    admin: 3,
    editor: 50,
    viewer: 100
  }
})
```

## Obtener Balance

```typescript
const balance = await commet.seats.getBalance({
  externalId: 'user_123',
  seatType: 'editor'
})

console.log(`Seats actuales: ${balance.data.current}`)
```

## Obtener Todos los Balances

```typescript
const balances = await commet.seats.getAllBalances({
  externalId: 'user_123'
})

console.log(balances.data)
// { editor: { current: 50, asOf: '...' }, viewer: { current: 100, asOf: '...' } }
```

## Identificación de Cliente

Usa `customerId` o `externalId`:

```typescript
// Por ID de Commet
await commet.seats.add({
  customerId: 'cus_xxx',
  seatType: 'editor',
  count: 5
})

// Por ID de tu sistema
await commet.seats.add({
  externalId: 'stripe_cus_abc123',
  seatType: 'editor',
  count: 5
})
```
