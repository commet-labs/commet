<Callout type="info">
**Prerrequisitos:** El cliente debe existir y tener tipos de seat configurados.
</Callout>

## Agregar Seats

```typescript
const result = await commet.seats.add({
  customerId: 'cus_xxx',
  seatType: 'editor',
  count: 5
})
```

<Callout type="info">
**Identificación de Cliente**: Usa `customerId` o `externalId` para identificar clientes.
</Callout>

## Remover Seats

```typescript
const result = await commet.seats.remove({
  customerId: 'cus_xxx',
  seatType: 'editor',
  count: 2
})
```

## Establecer Cantidad Exacta

```typescript
const result = await commet.seats.set({
  customerId: 'cus_xxx',
  seatType: 'editor',
  count: 10
})
```

## Usando IDs Externos

```typescript
const result = await commet.seats.add({
  externalId: 'stripe_cus_abc123',
  seatType: 'editor',
  count: 5
})
```

## Actualización Masiva

Actualiza múltiples tipos de seat a la vez:

```typescript
const result = await commet.seats.bulkUpdate({
  customerId: 'cus_xxx',
  seats: {
    admin: 3,
    editor: 50,
    viewer: 100
  }
})
```

## Obtener Balance

```typescript
const balance = await commet.seats.getBalance({
  customerId: 'cus_xxx',
  seatType: 'editor'
})

console.log(`Seats actuales: ${balance.data.current}`)
```

## Obtener Todos los Balances

```typescript
const allBalances = await commet.seats.getAllBalances({
  customerId: 'cus_xxx'
})

console.log(allBalances.data)
// { editor: { current: 50, asOf: '...' }, viewer: { current: 100, asOf: '...' } }
```

## Listar Eventos

```typescript
const events = await commet.seats.listEvents({
  customerId: 'cus_xxx',
  seatType: 'editor',
  limit: 50
})

// Paginación
if (events.hasMore) {
  const nextPage = await commet.seats.listEvents({
    customerId: 'cus_xxx',
    cursor: events.nextCursor
  })
}
```
