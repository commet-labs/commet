---
title: Planes
description: Listar y obtener planes de pricing disponibles
---

Los planes se configuran en el [dashboard](https://commet.co). Usa el SDK para listarlos u obtener detalles.

## Listar Planes

```typescript
const plans = await commet.plans.list()
```

Incluir planes privados:

```typescript
const plans = await commet.plans.list({ includePrivate: true })
```

## Obtener Plan

```typescript
const plan = await commet.plans.get('pro')
```
