# CLI de Commet

Autenticar, vincular proyectos y generar tipos TypeScript desde tu configuración de Commet.

## Inicio Rápido

```bash
# Instalar
npm install -g commet

# Autenticar
commet login

# Vincular a organización
commet link

# Generar tipos
commet pull
```

## Qué Hace

**Autenticación**: Flujo de dispositivo OAuth con entornos sandbox y producción.

**Vinculación de Proyectos**: Crea archivo `.commet` con configuración de organización.

**Generación de Tipos**: Genera `.commet/types.d.ts` con tus event types, seat types, planes y features para autocompletado en tu código.

## Comandos

### Autenticación
- `commet login` - Autenticar con Commet
- `commet logout` - Eliminar credenciales
- `commet whoami` - Mostrar estado de autenticación

### Gestión de Proyectos
- `commet link` - Vincular proyecto a organización
- `commet unlink` - Desvincular proyecto
- `commet switch` - Cambiar organización
- `commet info` - Mostrar detalles del proyecto

### Generación de Tipos
- `commet pull` - Generar tipos TypeScript
- `commet list events` - Listar event types
- `commet list seats` - Listar seat types

Ver [Comandos](/docs/library/cli/commands) para referencia completa.

## Configuración

**`~/.commet/auth.json`**: Credenciales de autenticación globales

**`.commet`**: Configuración de organización del proyecto

**`.commet/types.d.ts`**: Tipos TypeScript generados
