## Autenticaci√≥n

### `commet login`

Autenticar con Commet usando flujo OAuth.

```bash

```

Solicita el entorno (Sandbox/Producci√≥n), abre el navegador, almacena credenciales en `~/.commet/auth.json`.

---

### `commet logout`

```bash
commet logout
```

Elimina `~/.commet/auth.json`.

---

### `commet whoami`

```bash

```

Muestra el estado de autenticaci√≥n y la organizaci√≥n vinculada.

## Generaci√≥n de Tipos

### `commet pull`

Generar tipos TypeScript en `.commet/types.d.ts`.

```bash
commet pull
```

Crea:
- Directorio `.commet/`
- `.commet/types.d.ts` con event types, seat types y products
- Actualiza `tsconfig.json` para incluir los tipos
- Actualiza `.gitignore` para ignorar `.commet/`

**Archivo generado:**
```typescript
// .commet/types.d.ts
/**
 * Event types disponibles en tu organizaci√≥n:
 * - "api_call": API Calls - Seguimiento de uso de API
 * - "email_sent": Emails Sent - Eventos de entrega de email
 * 
 * Seat types disponibles en tu organizaci√≥n:
 * - "admin": Administrator - Acceso completo (Free)
 * - "user": Standard User - Acceso regular
 * 
 * Products disponibles en tu organizaci√≥n:
 * - "prd_c2JV8SjRxSD2y6cRSMzYJPJ2": API Platform - Facturaci√≥n por uso
 * - "prd_fnvJZwETRrYTmt9y42tLQEMH": Support Plan - Tarifa mensual fija
 */
declare module '@commet/node' {
  interface CommetGeneratedTypes {
    eventType: "api_call" | "email_sent";
    seatType: "admin" | "user";
    productId: "prd_c2JV8SjRxSD2y6cRSMzYJPJ2" | "prd_fnvJZwETRrYTmt9y42tLQEMH";
  }
}
```

**Uso:**
```typescript
import { Commet } from '@commet/node';

const commet = new Commet({ apiKey: '...' });

// Autocompletado para event types
await commet.usage.create({
  customerId: 'cus_123',
  eventType: 'api_call', // ‚úì autocompletado
});

// Autocompletado para seat types
await commet.seats.add({
  customerId: 'cus_123',
  seatType: 'admin', // ‚úì autocompletado
  count: 5
});

// Autocompletado para products
await commet.subscriptions.create({
  customerId: 'cus_123',
  productId: 'prd_c2JV8SjRxSD2y6cRSMzYJPJ2', // ‚úì autocompletado
});
```
