# Flujos de Trabajo del CLI

## Configuración Inicial

<Steps>
<Step>

### Instalar y Autenticar

```bash
npm install -g commet
commet login
```

</Step>

<Step>

### Vincular y Generar Tipos

```bash
cd tu-proyecto
commet link
commet pull
```

</Step>
</Steps>

---

## Desarrollo Diario

Después de agregar event/seat types en el dashboard:

```bash
commet pull
```

Tu código ahora tiene autocompletado para los nuevos tipos.

---

## Cambiar Organizaciones

```bash
cd proyecto-a
commet switch  # Seleccionar Organización A

cd proyecto-b
commet switch  # Seleccionar Organización B
```

---

## Cambiar Entornos

```bash
commet logout
commet login  # Seleccionar entorno diferente
commet link
commet pull
```

---

## Sincronización de Equipo

**Opción 1**: Commitear `.commet/types.d.ts` a git

Los miembros del equipo obtienen los tipos automáticamente desde git.

**Opción 2**: Cada desarrollador ejecuta `commet pull`

Agregar `.commet/` a `.gitignore`.
