# Instalando el CLI de Commet

Requiere Node.js 18+.

## Instalar

<Tabs items={['npm', 'pnpm', 'yarn']}>
<Tab value="npm">
```bash

```
</Tab>
<Tab value="pnpm">
```bash

```
</Tab>
<Tab value="yarn">
```bash

```
</Tab>
</Tabs>

