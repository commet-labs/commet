Sandbox es el entorno por defecto - completamente aislado de producción.

## Tarjetas de Prueba

**Pagos exitosos:**

| Tarjeta | Marca |
|------|-------|
| `4242 4242 4242 4242` | Visa |
| `5555 5555 5555 4444` | Mastercard |

Usa cualquier fecha de vencimiento futura y cualquier CVC de 3 dígitos.

**Pagos fallidos:**

| Tarjeta | Escenario |
|------|----------|
| `4000 0000 0000 9995` | Fondos insuficientes |
| `4000 0000 0000 0002` | Tarjeta declinada |

**3D Secure:**

| Tarjeta | Descripción |
|------|-------------|
| `4000 0025 0000 3155` | Requiere autenticación |

## Flujo de Prueba

```typescript
// 1. Crear cliente
const customer = await commet.customers.create({
  email: 'test@example.com',
  externalId: `test_${Date.now()}`,
})

// 2. Crear suscripción
const subscription = await commet.subscriptions.create({
  externalId: customer.data.externalId,
  planCode: 'pro',
})

// 3. Pagar con tarjeta de prueba en checkoutUrl

// 4. Rastrear uso
await commet.usage.track({
  externalId: customer.data.externalId,
  eventType: 'api_call',
})
```

## Producción

```typescript
const commet = new Commet({
  apiKey: process.env.COMMET_PRODUCTION_KEY!,
  environment: 'production',
})
```
