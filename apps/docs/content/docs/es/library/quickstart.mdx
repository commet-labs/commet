# Inicio Rápido

Pon tu sistema de facturación en funcionamiento en 5 minutos.

<Callout type="info">
**Prerrequisitos:** Node.js 18+ y una [cuenta de Commet](https://commet.co).
</Callout>

<Steps>
<Step>

## Instalar

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
<Tab value="npm">
```bash
npm install @commet/node
```
</Tab>
<Tab value="pnpm">
```bash
pnpm add @commet/node
```
</Tab>
<Tab value="yarn">
```bash
yarn add @commet/node
```
</Tab>
<Tab value="bun">
```bash
bun add @commet/node
```
</Tab>
</Tabs>

</Step>

<Step>

## Inicializar

```typescript
import { Commet } from '@commet/node'

const commet = new Commet({
  apiKey: process.env.COMMET_API_KEY!,
  environment: 'sandbox', // o 'production'
})
```

</Step>

<Step>

## Rastrear Uso

```typescript
await commet.usage.create({
  eventType: 'api_call',
  customerId: 'cus_xyz',
  idempotencyKey: `evt_${Date.now()}`,
  properties: [
    { property: 'endpoint', value: '/users' },
    { property: 'method', value: 'GET' }
  ]
})
```

</Step>

<Step>

## Gestionar Seats

```typescript
// Agregar seats
await commet.seats.add({
  customerId: 'cus_xyz',
  seatType: 'editor',
  count: 5
})

// Establecer cantidad exacta
await commet.seats.set({
  customerId: 'cus_xyz',
  seatType: 'editor',
  count: 10
})
```

</Step>

<Step>

## Crear Suscripción

```typescript
const subscription = await commet.subscriptions.create({
  customerId: 'cus_xyz',
  items: [
    { priceId: 'price_xxx', quantity: 1 }
  ]
})
```

</Step>
</Steps>
