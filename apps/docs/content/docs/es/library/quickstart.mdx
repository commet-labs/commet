# Inicio Rápido

Pon tu sistema de facturación en funcionamiento en 5 minutos.

<Callout type="info">
**Prerrequisitos:** Node.js 18+ y una [cuenta de Commet](https://commet.co).
</Callout>

<Steps>
<Step>

## Instalar

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
<Tab value="npm">
```bash
npm install @commet/node
```
</Tab>
<Tab value="pnpm">
```bash
pnpm add @commet/node
```
</Tab>
<Tab value="yarn">
```bash
yarn add @commet/node
```
</Tab>
<Tab value="bun">
```bash
bun add @commet/node
```
</Tab>
</Tabs>

</Step>

<Step>

## Inicializar

```typescript
import { Commet } from '@commet/node'

const commet = new Commet({
  apiKey: process.env.COMMET_API_KEY!,
  environment: 'sandbox', // o 'production'
})
```

</Step>

<Step>

## Suscribir a un Plan

Cada cliente puede tener una suscripción activa a la vez.

```typescript
// Suscribir cliente a un plan
const subscription = await commet.subscriptions.create({
  externalId: 'user_123',  // Tu user ID
  planId: 'plan_pro',
  billingInterval: 'monthly',
  initialSeats: { editor: 5 }
})
```

</Step>

<Step>

## Rastrear Uso

```typescript
await commet.usage.track({
  externalId: 'user_123',
  eventType: 'api_call',
  idempotencyKey: `evt_${Date.now()}`,
  properties: { endpoint: '/users', method: 'GET' }
})
```

</Step>

<Step>

## Gestionar Seats

```typescript
// Agregar seats
await commet.seats.add({
  externalId: 'user_123',
  seatType: 'editor',
  count: 5
})

// Establecer cantidad exacta
await commet.seats.set({
  externalId: 'user_123',
  seatType: 'editor',
  count: 10
})
```

</Step>
</Steps>
